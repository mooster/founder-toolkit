Claude Code Plugin · Open Source

Skills for founders,
not just developers.

Investor updates, pitch reviews, SaaS metrics, and lead scoring — built from real startup operating experience.

Install in Claude Code
/plugin marketplace add mooster/founder-toolkit
Then: /plugin install founder-toolkit@mooster-founder-toolkit

4 Skills, Zero Configuration

✉

Investor Update Writer

"write an investor update"

Input your MRR, burn, and highlights. Get a concise, professional update following YC and a16z best practices. Under 500 words, no fluff.

★

Pitch Deck Reviewer

"review my pitch deck"

10-dimension scoring from a VC perspective. Narrative, market size, traction, team, the ask — with specific feedback and red flags.

▢

Startup Metrics Dashboard

"calculate our SaaS metrics"

Full metric suite from raw numbers: ARR, NRR, LTV:CAC, burn multiple, runway, magic number. With stage-appropriate benchmarks.

▶

Outreach Lead Scorer

"score these leads"

Define your ICP, input a company list. Get A/B/C/D grades with transparent rationale. Based on a system that scored 300+ companies.
