What I believe

Three ideas from running a company through the AI transition. These drive everything below.

  • Intelligence is no longer scarce. Direction is.

    Analysis, coding, writing, pattern recognition — all abundant now. What's scarce: knowing which direction to point it. Context, judgment, and the willingness to own the outcome.

  • AI amplifies direction. It doesn't produce it.

    Give it good direction, 100x leverage. Bad direction, 100x speed into a wall. No direction, noise. The CEO's job isn't to do cognitive work — it's to set the vector.

  • Build position, not service.

    Selling cognitive work gets compressed by AI. Selling direction — frameworks, judgment, authority — gets amplified. Andreessen built Netscape first, then wrote the manifesto.
